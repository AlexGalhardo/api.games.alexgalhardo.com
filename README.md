## Introduction

## Development Setup Local

*   Clone repository

<!---->

    git clone git@github.com:AlexGalhardo/api.nerdapi.com.git

*   Enter repository

<!---->

    cd api.nerdapi.com/

*   Install dependencies

<!---->

    npm install

*   Start local server

<!---->

    npm run dev

*   Go to: <http://localhost:3000/>

## Before Submit Commits & PRs

<!---->
